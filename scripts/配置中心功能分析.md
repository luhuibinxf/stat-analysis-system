# 配置中心功能分析

## 界面分析

### 功能菜单
1. **用户配置和维护** - 用户管理功能
2. **权限维护** - 权限管理功能
3. **参数配置** - 系统参数配置
4. **每日分析** - 数据统计分析
5. **复杂存储配置** - 存储配置管理

### 用户管理功能
- **添加用户** - 绿色按钮
- **编辑用户** - 蓝色按钮
- **删除用户** - 红色按钮
- **用户列表** - 包含以下字段：
  - 用户ID (数字类型)
  - 用户名
  - 角色 (如：普通用户)
  - 状态 (如：启用)
  - 操作 (编辑、删除按钮)

## 逻辑分析

### 1. 用户管理逻辑

#### 添加用户
- 输入用户ID（数字类型）
- 输入用户名
- 输入密码
- 选择角色
- 设置状态（启用/禁用）
- 保存到数据库

#### 编辑用户
- 选择要编辑的用户
- 修改用户名
- 修改密码（可选）
- 修改角色
- 修改状态
- 保存修改

#### 删除用户
- 选择要删除的用户
- 确认删除操作
- 从数据库中删除用户

#### 查看用户列表
- 从数据库查询所有用户
- 显示用户ID、用户名、角色、状态
- 提供编辑和删除操作

### 2. 权限维护逻辑
- 角色管理
- 权限分配
- 权限验证

### 3. 参数配置逻辑
- 系统参数设置
- 数据库连接配置
- 其他系统配置

### 4. 每日分析逻辑
- 数据统计
- 报表生成
- 趋势分析

### 5. 复杂存储配置逻辑
- 存储路径配置
- 存储策略设置
- 备份配置

## 技术实现

### 数据库表结构
- **TJYHB** - 用户表
  - YHID - 用户ID (数字类型)
  - YHMC - 用户名
  - YKL - 密码
  - JSID - 角色ID
  - ZT - 状态 (1=启用, 0=禁用)
  - IS_DEL - 删除标记

### 实现方式
- PowerShell脚本实现基本功能
- SQL脚本实现数据查询和操作
- 批处理文件提供交互界面
